### üìã Project Summary

This project involved comprehensive testing of the SauceDemo application, encompassing both **functional UI testing** and **back-end API testing**. The goal was to establish a thorough baseline for application quality, identify critical defects, and formalize all QA processes.

| Metric | Details |
| :--- | :--- |
| **Total Test Cases** | 34 (Covering all key features: Login, Inventory, Cart, Checkout, etc.) |
